# Live Coding
Codigo feito em nodejs durante o live coding do desafio, desafio consistia em uma lista de instruções para certos tipos de cenários

> [!NOTE]
> Feito em 1 Hora. da forma mais simples e legível

### Iniciando
- Instale nodejs em seu dispositivo
- Entre na pasta ``livecoding``: ``cd livecoding``
- npm install
- node index.js

# Demonstração PHP Laravel

> [!NOTE]
> Esse é um projeto derivado de um desafio e não deve ser usado em produção.

### Iniciando
- Instale php e composer no seu OS

``Extensões Necessárias do PHP``
```

extension=pdo_sqlite
extension=sqlite3
extension=openssl
extension=fileinfo
extension=mbstring
extension=zip
```
- Clone o repositorio atual ``git clone https://github.com/LeandroTheDev/motorista_px_challenge``
- Entre na pasta ``demonstration``: ``cd demonstration``
- Instale dependenciais do composer: ``composer install``
- Gere uma chave: ``php artisan key:generate``
- Inicie o servidor: ``php artisan serve``
